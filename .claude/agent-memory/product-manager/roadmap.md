# Product Roadmap

## Phase 1: Core Editing

**Outcome**: User can visually build any static UI and get the exact same thing as React code.

**Completion criteria**:

- Any UI layout can be constructed visually in this editor
- The generated code renders identically to what the editor shows
- All styling options are editable visually

---

## Phase 2: Component System

**Outcome**: User can turn a piece of UI into a reusable component and place it multiple times. Editing the component updates all instances.

**Completion criteria**:

- User can create a component, place multiple instances, and edit it in one place to update all
- Instances can override specific properties without breaking the link
- Components have editable props

---

## Phase 3: Design System

**Outcome**: User can define design tokens and apply them across the project. Changing a token updates everywhere it's used.

**Completion criteria**:

- User can define tokens, apply them to elements, change the value, and see it update everywhere
- Tokens are reflected in the generated code

---

## Phase 4: Plugin System & AI Editing

**Outcome**: User can extend the editor with plugins and use AI to assist editing.

**Completion criteria**:

- Plugin system allows third-party extensions to the editor
- AI can assist with editing tasks within the editor
